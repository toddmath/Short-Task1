WURFL Capabilities

advertised_browser
Shows the name of the device browser, based off of it's advertised user agent.
is_wireless_device
Boolean, that returns true when the device is found to have wireless(3G, LTE) capabilities. is_mobile is an alias to this, as well.
max_image_width
Returns the maximum width in pixels, the device can view on it's screen. It automatically defaults to mobile viewports specified inside meta tags, if present.
